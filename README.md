# Life Organiser PWA

A simple Progressive Web App for managing:

- Tasks
- Routines
- Habits
- Important dates
- Work and personal items

## How to run locally

Open the folder in VS Code and use a local server.

With Python:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## How to put it on GitHub

```bash
git add .
git commit -m "Add Life Organiser PWA"
git push
```

## How to publish with GitHub Pages

1. Go to your GitHub repository.
2. Open Settings.
3. Open Pages.
4. Choose Deploy from branch.
5. Choose main branch and root folder.
6. Save.

After a short time, GitHub gives you a website link.

## How to install on Samsung phone

1. Open the website in Chrome.
2. Tap the menu.
3. Tap Add to Home screen or Install app.
